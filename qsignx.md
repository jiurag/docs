## 安装jdk环境

::: code-group

```sh [centos]
#检查yum中是否有jdk1.8的安装包
yum list java-1.8*
#安装
yum install java-1.8.0-openjdk* -y
#查询版本
java -version
#输出以下信息则安装完成
openjdk version "1.8.0_222"
OpenJDK Runtime Environment (build 1.8.0_222-b10)
OpenJDK 64-Bit Server VM (build 25.222-b10, mixed mode)
```
