## 安装jdk环境

::: code-group

``` [centos]
#检查yum中是否有jdk1.8的安装包
yum list java-1.8*
#安装
yum install java-1.8.0-openjdk* -y
```

``` [Ubuntu]
#任选一种版本安装
#安装jdk8
apt install openjdk-8-jdk
#安装jdk11
apt install openjdk-11-jdk
#安装jdk17
apt install openjdk-17-jdk
```

:::

**查询java版本**

```
java -version
```
